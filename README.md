🍰 Вкусняшка — интернет-магазин кондитерских изделий
«Вкусняшка» — это веб-приложение для продажи кондитерских изделий через интернет. Проект разработан в рамках курсовой работы по дисциплине «Технология разработки программного обеспечения» (траектория Б: Django REST + React SPA). Приложение позволяет пользователям просматривать каталог сладостей, добавлять товары в корзину, оформлять заказы, отслеживать историю покупок, управлять профилем, а администраторам — управлять товарами, категориями и заказами.

Технологический стек
Backend
Django — веб-фреймворк

Django REST Framework (DRF) — создание REST API

Simple JWT — JWT-аутентификация (access/refresh токены)

SQLite (разработка) 

django-cors-headers — настройка CORS

django-filter — фильтрация и поиск товаров

Pillow — обработка изображений (аватары, фото товаров)

Frontend
React — библиотека для построения интерфейсов

Create React App (CRA) — сборка проекта

React Router — маршрутизация

Axios — HTTP-клиент (перехватчики для JWT)

CSS Modules + custom CSS — стилизация в розовой тематике

Основной функционал
Публичная часть (доступна всем)
Просмотр каталога товаров (с пагинацией)
Просмотр карточки товара (описание, цена, фото)
Поиск товаров по названию
Фильтрация товаров по категориям (торты, пирожные, печенье, эклеры и др.)
Сортировка товаров (по цене, новинки)
Регистрация и аутентификация (JWT)
Приватная часть (для авторизованных пользователей)
Добавление товаров в корзину
Изменение количества товаров в корзине
Удаление товаров из корзины
Оформление заказа (адрес доставки, телефон)
Просмотр истории заказов
Редактирование профиля (имя, email, телефон, «о себе»)
Административная часть (для администраторов)
Управление товарами (CRUD: создание, редактирование, удаление)
Управление категориями товаров
Управление пользователями
Управление заказами (изменение статуса)

Установка и запуск
Требования
Python 3.10+
Node.js 18+
Открыть в браузере
Фронтенд: http://localhost:3000

Админка Django: http://localhost:8000/admin

API (browsable): http://localhost:8000/api/products/

📚 Документация API
После запуска бэкенда API доступно по адресу: http://localhost:8000/api/

Основные эндпоинты
Метод	Эндпоинт	Описание
POST	/api/auth/register/	Регистрация
POST	/api/auth/login/	Получение JWT (access/refresh)
POST	/api/auth/token/refresh/	Обновление access-токена
GET/PATCH	/api/auth/profile/	Профиль пользователя
GET	/api/categories/	Список категорий
GET	/api/products/	Список товаров (пагинация, поиск, фильтрация)
GET	/api/products/{id}/	Детали товара
GET	/api/cart/	Корзина пользователя
POST	/api/cart/add/	Добавить товар в корзину
PATCH/DELETE	/api/cart/item/{id}/	Изменить количество / удалить
GET	/api/orders/	История заказов
POST	/api/orders/create/	Оформить заказ
PUT	/api/orders/{id}/status/	Изменить статус (админ)
