# 🍰 Вкусняшка — интернет-магазин кондитерских изделий

Веб-приложение для продажи кондитерских изделий через интернет. Разработано в рамках курсового проекта по дисциплине «Технология разработки программного обеспечения» (траектория Б: Django REST + React SPA).



## 📌 Описание проекта

«Вкусняшка» — это полнофункциональный интернет-магазин, который позволяет пользователям просматривать каталог сладостей, добавлять товары в корзину, оформлять заказы, отслеживать историю покупок и управлять профилем. Администраторы могут управлять товарами, категориями и заказами.

### Основной функционал

- **Для гостей:** просмотр каталога, регистрация, вход
- **Для авторизованных пользователей:** корзина, оформление заказов, история заказов, профиль
- **Для администраторов:** управление товарами, категориями, заказами, пользователями

## 🛠 Установка и запуск

### Требования

- Python 3.10+
- Node.js 18+
- npm

### 1. Клонирование репозитория

```bash
git clone https://github.com/LeraRepchenko/ConditerShop.git
cd ConditerShop
2. Настройка бэкенда
bash
cd backend
python -m venv venv
# Windows: venv\Scripts\activate
# macOS/Linux: source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
Бэкенд будет доступен по адресу: http://localhost:8000

3. Настройка фронтенда
bash
cd frontend
npm install
npm start
Фронтенд будет доступен по адресу: http://localhost:3000

4. API-документация
После запуска бэкенда документация API доступна по адресам:

Swagger UI: http://localhost:8000/api/docs/

ReDoc: http://localhost:8000/api/schema/redoc/

📋 API-эндпоинты (основные)
Метод	Эндпоинт	Назначение	Доступ
POST	/api/auth/register/	Регистрация	Все
POST	/api/auth/login/	Вход (JWT)	Все
POST	/api/auth/token/refresh/	Обновление access токена	Все
GET	/api/products/	Список товаров	Все
GET	/api/categories/	Список категорий	Все
GET	/api/cart/	Корзина	Авторизованные
POST	/api/orders/create/	Оформить заказ	Авторизованные
GET	/api/orders/	История заказов	Авторизованные
Полный список — в Swagger-документации.
